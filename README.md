### 소감, 어려움, 또는 건의사항
사실 한 번도 JAVA를 코딩해본 적이 없어서 객체 지향 프로그래밍을 접해본 경험이 적다. 그래서 더 객체 지향 프로그래밍을 더 공부할 필요를 느꼈다. 
그리고 코딩을 하면서 다양한 IDE를 경험해 볼 수 있어서 좋았다. 점점 내가 성장하고 있다는 것을 알게 되는 거 같아서 감사했다 :) 
더욱 더 열심히 공부해서 온전히 이해할 수 있도록 노력해야겠다.
